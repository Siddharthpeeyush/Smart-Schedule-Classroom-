## 🎓 Smart Classroom & Timetable Scheduler v3.0

A complete college timetable management system built with Python (Flask) 
and vanilla HTML/CSS/JS — delivered as a single-file web application.

### 🚀 Features
- 🔐 Role-based login (Admin & Faculty)
- 📅 Auto timetable generation with conflict detection
- 🏫 Multi-department, multi-batch support
- 📚 Electives & elective group management
- 🏖 Faculty leave & substitution system
- 📢 Notice board with priority levels
- 🤖 AI scheduling chat assistant
- 🖱 Drag-to-swap timetable slots
- 📊 Live Chart.js analytics (4 charts)
- 📥 CSV export with institution header
- 🗄 Semester archive (save & restore)
- ⚠️ Workload overload detection
- 🌙 Dark / Light theme toggle

### 🛠 Tech Stack
- Backend  : Python, Flask
- Frontend : HTML, CSS, JavaScript (single-file)
- Charts   : Chart.js
- Auth     : Session-based with MD5 hashing
- Storage  : In-memory (no database required)

### ▶️ Run Locally
pip install flask
python app.py
Open → http://localhost:5000
